删除存储过程和函数
===

当某个存储过程或函数不再需要，或者准备重建时，可以将其删除。

### 删除存储过程

删除存储过程的语法如下：

```sql
DROP PROCEDURE 过程名;
```

例如：

```sql
DROP PROCEDURE get_all_user;
```

为了避免对象不存在时报错，也可以使用：

```sql
DROP PROCEDURE IF EXISTS get_all_user;
```

### 删除存储函数

删除存储函数的语法如下：

```sql
DROP FUNCTION 函数名;
```

例如：

```sql
DROP FUNCTION get_level;
```

同样也可以写成：

```sql
DROP FUNCTION IF EXISTS get_level;
```

### 小结

删除存储过程和函数的语法都很直接，核心区别只是对象类型不同。真正要注意的是删除前确认是否还有应用代码、脚本或其他对象依赖它们。
