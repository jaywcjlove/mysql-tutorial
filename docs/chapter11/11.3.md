查看视图
===

创建视图后，通常需要查看其结构、定义和元信息。MySQL 提供了多种查看视图的方式。

### DESCRIBE语句查看视图基本信息

可以直接使用 `DESCRIBE` 或 `DESC`：

```sql
DESCRIBE user_simple;
```

它会显示视图的字段信息，例如：

- 字段名
- 类型
- 是否允许为空

### SHOW TABLE STATUS语句查看视图基本信息

还可以使用：

```sql
SHOW TABLE STATUS LIKE 'user_simple';
```

这可以查看一些基础状态信息，例如对象名称、类型等。

### SHOW CREATE VIEW语句查看视图详细信息

如果要查看视图的完整定义，最直接的方式是：

```sql
SHOW CREATE VIEW user_simple;
```

这个结果会返回完整的创建视图 SQL。

### 在views表中查看视图详细信息

也可以通过 `information_schema.VIEWS` 查看视图信息：

```sql
SELECT TABLE_NAME, VIEW_DEFINITION
FROM information_schema.VIEWS
WHERE TABLE_SCHEMA = DATABASE();
```

这种方式适合批量查看多个视图。

### 小结

想快速看字段可以用 `DESCRIBE`，想看完整定义可以用 `SHOW CREATE VIEW`，想批量筛选和脚本化处理则更适合使用 `information_schema.VIEWS`。
